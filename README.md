# Basic-Image-Classification-with-TensorFlow

## Project Overview

This repository contains a guided project completed through Coursera, focused on building a basic image classification model using **Keras** with **TensorFlow** as the backend.

Throughout this project, I learned the fundamentals of **Artificial Neural Networks (ANNs)** and implemented a complete machine learning workflow including data preprocessing, model creation, training, evaluation, and prediction visualisation.

The final model was developed to predict digits from handwritten images with high accuracy.

## Project Workflow

This guided project covered the following steps:

### 1. Encode Labels

Prepared target labels into a machine-readable format.

### 2. Understand Neural Networks

Learned the basic concepts and architecture of neural networks.

### 3. Preprocess Image Data

Prepared and normalised image examples before training.

### 4. Build the Neural Network

Created a classification model using Keras and TensorFlow.

### 5. Train the Model

Fitted the model to the training dataset.

### 6. Evaluate Performance

Measured model performance and prediction accuracy.

### 7. Visualize Predictions

Generated and interpreted prediction outputs.
